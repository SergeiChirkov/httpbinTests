# httpbinTests

все что нужно для запуска:
1) открыть проект как maven project
2) java 1.8
3) создать папку Logs вручную в корне проекта (потому что автоматически она создаваться с slf4j не стала, а разбираться мне не захотелось)
4) перейти в src/test/java/Runner/Runner.java
5) установить тэг @Regression если нужно запустить все тесты или любой другой по желанию
6) запустить Runner.java

отчет сформируется в папку target/cucumber-html-reports
1) открыть файл overview-features.html

Лог будет в папке Logs/logfile.log (да, при каждом запуске перезаписывается =))
